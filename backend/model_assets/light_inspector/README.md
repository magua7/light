# 本地模型资源说明

该目录用于放置本地真实模型所需的大体积资源文件。

请将以下文件保留在本地，不要直接提交到 GitHub：

- `best2.pt`
- `score_total.tif`
- `cover_huan1.tif`

说明：

- 这些文件体积较大，其中 `cover_huan1.tif` 超过 GitHub 普通仓库单文件 100MB 限制。
- 项目运行时会默认从当前目录读取这些资源。
- 建议通过线下拷贝、共享网盘或部署脚本同步到目标机器。

当前目录下的小型文本资源仍可保留在仓库中：

- `广告牌.txt`
- `上射光.txt`
- `路灯.txt`
- `车灯.txt`
